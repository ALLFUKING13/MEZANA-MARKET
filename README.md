# Mezana Market - Telegram Mini App

Mezana Market is a modern, glassmorphic Telegram Mini App for online shopping. It features a catalog of products, a cart system, and a checkout process.

## Features

- **Glassmorphic UI**: Beautiful and modern aesthetics.
- **Product Catalog**: Categorized products with detailed information.
- **Cart System**: Easy-to-use cart for managing selected items.
- **Multi-language Support**: Uzbek, Russian, Korean, and English.
- **Telegram Integration**: Seamless integration with the Telegram Mini App platform.

## How to Deploy to GitHub Pages

1. **Create a new repository** on GitHub.
2. **Push these files** to your repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - Go to your repository **Settings** on GitHub.
   - Click on **Pages** in the left sidebar.
   - Under **Build and deployment**, select the **main** branch and **(root)** folder.
   - Click **Save**.
4. Your site will be available at `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`.

## How to Deploy to Vercel (Alternative)

1. Go to [vercel.com](https://vercel.com/) and Sign Up/Log In using your **GitHub** account.
2. Click on **"Add New"** > **"Project"**.
3. Import your repository from GitHub.
4. Vercel will automatically detect that this is a static site. No configuration is needed.
5. Click **"Deploy"**.
6. Your site will be live at `https://YOUR_REPO_NAME.vercel.app/`.
